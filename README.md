# Git, Github e VSCODE: 

O GitHub é uma plataforma online para armazenar e colaborar em projetos de programação. Ele usa o sistema Git para manter o controle das alterações no código, o que ajuda a gerenciar diferentes versões e facilitar o trabalho em equipe, auxiliando e facilitando a vida dos programadores. Dica: Use sempre o site em ingles, para adaptação.
 # Funcionalidades: 
Branch: 
No GitHub, uma "branch" (ou ramificação) é uma versão paralela do código dentro de um repositório.  

Repositório, diretório ou pasta: 
Todos esses termos possuem o mesmo significado. Um repositório, no contexto do GitHub e do Git em geral, é um local onde o código-fonte e todos os arquivos associados a um projeto são armazenados e gerenciados. 

Pull request: 
Um pull request é uma solicitação para integrar alterações feitas em uma branch de um repositório no GitHub (ou em outras plataformas de controle de versão) à branch principal ou a outra branch dentro do mesmo repositório. 

Git merge: 
O principal objetivo do git merge é combinar o trabalho feito em diferentes branches. Por exemplo, você pode estar trabalhando em uma branch de desenvolvimento e querer incorporar essas mudanças na branch principal. 

  # Readme.md, o que é? 
O arquivo README.md no GitHub é um documento crucial para qualquer repositório de código. Ele é usado para fornecer informações e orientações sobre o projeto. 
 # Como criar um repositório? 
 + No seu perfil no Github, clique em "your repositories" , depois no botão "new".
 + Na página de criação, determine um nome ao seu repositório, e também o nível de privacidade (privado ou público). 
 + Clique em " add a README file ". Por fim, em "create repository". Seu repositório está criado.

# Variáveis
+ var é a forma mais antiga de definir variáveis no javascript. Diferentemente da const e let,pode ter seu valor reatribuído e redeclarado. Ela não possue escopo de bloco.
+ let junto com a const vieram no ES6 (2015) , assim como a var pode ter seu valor reatribuído mas não, redeclaro.
+ const não pode ter seu valor reatribuído e nem declarado. Assim, sendo diferente de var e let. Também possui escopo de bloco.
  # rotulo 
 O rotulo funciona como uma caixa, e dentro vão os dados. Podendo ser eles, listas, dados etc.
 Dica: sempre use a palavra sem acento.
 Exemplos: 
     
     let rotulo = "Ana";
     console.log(rotulo);
     saída: Ana
     
# Strings
+ São a sequência de caracteres para representar texto. Basta utilizar aspas, crase ou apóstrofos no início e final da palavra, frase ou numeral.
  Exemplos:
  'Hello world' \ "Hello world" \ ´Hello world´
